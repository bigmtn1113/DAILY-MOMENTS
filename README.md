# DAILY-MOMENTS
서로의 일상을 게시물을 통해 공유하는 SNS 플랫폼

<br/>

## 프로젝트 목표
- Spring 기반 하이브리드 모바일 웹앱 개발
- 서로의 일상을 게시물을 통해 공유하는 SNS 플랫폼 개발
- 한국소프트웨어산업협회에서 배운 내용에 대한 이해와 응용

<br/>
